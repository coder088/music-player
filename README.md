#Questo progetto è un music player che permette di leggere file musicali inserendoli nella cartella src emodificando la stringa filepath con il nome del file. 
Eseguendo il programma si ha accesso a funzioni per fermare, resettare o far partire la musica oltre a due opzioni per alzare il volume.
Per realizzarlo non è stato fatto uso di intelligenza artificiale se non per aiutarmi nell'apprendimento del funzionamento della libreria javax.sound.sampled.FloatControl
considerando la scarsa quantità di risorse che ho trovato su internet. Il progetto è stato realizzato interamente da me nell'arco di una settimana.

#This project is a music player that allows you to play music files by inserting them into the src folder and modifying the filepath string with the file name.
Running the program provides access to functions for stopping, resetting, and starting the music, as well as two volume controls.
No artificial intelligence was used to develop it, except to help me learn how the javax.sound.sampled.FloatControl library works, given the limited resources I found online. I developed the project entirely myself in a week.
